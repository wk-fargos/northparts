# 🚀 Деплой NorthParts на Render.com
## Только 5 шагов, ~10 минут

---

## Шаг 1 — Создай аккаунт на GitHub (если нет)
👉 https://github.com/signup
Бесплатно. Нужен email + пароль.

---

## Шаг 2 — Загрузи проект на GitHub

1. Открой https://github.com/new
2. Repository name: `northparts`
3. Оставь **Public** (или Private — без разницы)
4. Нажми **Create repository**
5. На следующей странице нажми **uploading an existing file**
6. Распакуй архив `northparts_render.zip`
7. **Перетащи ВСЕ файлы из папки** в браузер:
   ```
   northparts/         ← всю папку целиком
   allegro_parser.py   ← этот файл рядом
   render.yaml         ← этот файл рядом
   ```
8. Нажми **Commit changes**

---

## Шаг 3 — Создай аккаунт на Render.com
👉 https://render.com
Нажми **Get Started for Free** → Sign up with GitHub (удобнее всего)

---

## Шаг 4 — Задеплой одной кнопкой

1. В Render нажми **New → Blueprint**
2. Выбери свой репозиторий `northparts`
3. Render автоматически найдёт `render.yaml` и создаст:
   - ✅ Web Service (Flask сервер)
   - ✅ PostgreSQL база данных
   - ✅ Все переменные окружения
4. Нажми **Apply**

Подожди **3–5 минут** пока Render установит зависимости.

---

## Шаг 5 — Готово!

После деплоя Render покажет адрес типа:
```
https://northparts.onrender.com
```

- 🛍 **Магазин**: https://northparts.onrender.com
- ⚙️ **Админка**: https://northparts.onrender.com/admin
- 🔐 **Логин**: admin / admin123

## ❓ Частые вопросы

**Сайт не открывается сразу?**
На бесплатном плане Render "засыпает" через 15 мин неактивности.
Первый запрос после сна занимает ~30 сек. Для теста — нормально.

**Как обновить сайт после изменений?**
Просто загрузи обновлённые файлы на GitHub — Render задеплоит автоматически.

**Хочу свой домен (например northparts.ca)?**
В Render → Settings → Custom Domains → добавь свой домен.
Бесплатный SSL-сертификат добавляется автоматически.

**База данных на бесплатном плане?**
PostgreSQL на Render Free: 1 GB, хватит на тысячи товаров и заказов.
Единственное ограничение — удаляется если нет активности 90 дней.
